# Shopify App Skills

Agent skills for Shopify app development with the latest stack (Remix/React Router, Prisma, Polaris).

## Install

```bash
npx skills add tamiror6/shopify-app-skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [shopify-admin-graphql](./skills/shopify-admin-graphql/) | Shopify Admin API GraphQL patterns - client setup, queries, throttling |
| [shopify-app-api-patterns](./skills/shopify-app-api-patterns/) | Frontend-backend communication in Shopify Remix apps |
| [shopify-built-for-shopify](./skills/shopify-built-for-shopify/) | Built for Shopify (BFS) quality standards and compliance |
| [shopify-polaris-web-components](./skills/shopify-polaris-web-components/) | Polaris Web Components for App Home UI |

## Tech Stack

These skills are designed for the modern Shopify app stack:

- **Framework**: [Remix](https://remix.run/) with `@shopify/shopify-app-remix`
- **Database**: [Prisma](https://www.prisma.io/)
- **UI**: [Polaris](https://polaris.shopify.com/) (React for embedded apps, Web Components for App Home)
- **API**: [Shopify Admin GraphQL API](https://shopify.dev/docs/api/admin-graphql)
